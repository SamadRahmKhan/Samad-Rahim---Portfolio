# Portfolio Website

## Project Overview

This is a personal portfolio website showcasing my work as a **UI Designer and Frontend Web Developer**.  
The site highlights selected projects, skills, experience, and includes a review/comment section for visitor feedback.

The focus of this project is clean UI, smooth animations, and a modern frontend architecture.

## Getting Started (Local Setup)

### Prerequisites

Make sure you have installed:
- Node.js
- npm

### Installation

```sh
# Clone the repository
git clone <https://samadrahmkhan.github.io/Samad-Rahim-Portfolio/#/>

# Go to project folder
cd <Samad-Rahim-Portfolio>

# Install dependencies
npm install

# Start development server
npm run dev
